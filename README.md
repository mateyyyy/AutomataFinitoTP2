# Trabajo Práctico Obligatorio N.º 2 – Autómatas Finitos

## Materias
- Sintaxis y Semántica de los Lenguajes  
- Teoría de la Computación

## Descripción
Este proyecto corresponde al segundo trabajo práctico obligatorio de la materia y tiene como objetivo aplicar los conocimientos adquiridos sobre **Autómatas Finitos**, un componente fundamental en la teoría de lenguajes formales y la computación.

El código desarrollado permite simular el comportamiento de **autómatas finitos deterministas (AFD)**. A partir de una definición formal del autómata (estados, alfabeto, estado inicial, estados finales y función de transición), el programa puede procesar cadenas de entrada y determinar si son aceptadas por el autómata.

## Funcionalidades
- Carga de un autómata finito determinista definido por el usuario.
- Procesamiento de cadenas para verificar si son aceptadas por el autómata.
- Visualización paso a paso de las transiciones del autómata (opcional, si aplica).
- Verificación de múltiples cadenas en una sola ejecución (si el código lo permite).
